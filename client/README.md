# Shadow Council

Shadow Council is a real-time multiplayer social deduction game where players join private rooms and compete using strategy, deception, and teamwork.

## Features

* User Authentication
* Real-time Multiplayer Gameplay
* Room Creation & Joining System
* Live Player Synchronization using Socket.IO
* Responsive React Frontend
* Node.js & Express Backend
* MongoDB Database Integration

## Tech Stack

### Frontend

* React
* Vite
* React Router
* Socket.IO Client
* Tailwind CSS

### Backend

* Node.js
* Express.js
* Socket.IO
* MongoDB
* JWT Authentication
